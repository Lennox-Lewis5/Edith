# EDITH Specification v1.0

## Name
EDITH

## Platform
Android

## Minimum Android Version
Android 10+

## Purpose
An AI assistant that helps users control their phone, answer questions, manage reminders, and automate daily tasks.

## Version 1 Features

- AI chat
- Voice commands
- Open apps
- Notes
- Reminders
- Battery information
- Dark futuristic interface

## Future Features

- Wake phrase
- Camera understanding
- Floating assistant
- Screen understanding
- Offline commands
